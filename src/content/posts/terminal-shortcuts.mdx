The most useful shortcuts for navigating and editing text in terminal.

## Navigation

|Keybinding|Action|
|---------|--------|
|<Kbd>Alt</Kbd> + <Kbd>f/b</Kbd>|Move cursor forward/backward one word|
|<Kbd>Ctrl</Kbd> + <Kbd>f/b</Kbd>|Move forward/backward one character|
|<Kbd>Ctrl</Kbd> + <Kbd>a/e</Kbd>|Move cursor to beginning/end of line|
|<Kbd>Ctrl</Kbd> + <Kbd>xx</Kbd>|Alternates the cursor with its old position|

## Editing

|Keybinding|Action|
|---------|--------|
|<Kbd>Alt</Kbd> + <Kbd>d</Kbd>|Delete next word|
|<Kbd>Alt</Kbd> + <Kbd>Del</Kbd>|Delete previous word|
|<Kbd>Ctrl</Kbd> + <Kbd>w</Kbd>|Delete previous word|
|<Kbd>Ctrl</Kbd> + <Kbd>d</Kbd>|Delete character under cursor|
|<Kbd>Ctrl</Kbd> + <Kbd>xu</Kbd>|Incremental undo|
|<Kbd>Ctrl</Kbd> + <Kbd>xe</Kbd>|Edit in the $EDITOR program|
|<Kbd>Ctrl</Kbd> + <Kbd>xv</Kbd>|Edit in vi mode|
|<Kbd>Ctrl</Kbd> + <Kbd>x</Kbd> + <Kbd>Ctrl</Kbd> + <Kbd>*</Kbd>|Expand `*` to all items in the current directory|
